Componente utilizado: Color animation

Enlace a la página del sitio oficial donde se muestra la demo: Aquí

Breve descripción: Permite alterar las propiedades del elemento durante la transición de colores.

Código:

$( function() { var state = true; $( "#button" ).on( "click", function() { if ( state ) { $( "#effect" ).animate({ backgroundColor: "#B24926", color: "#fff", width: 500 }, 3000 ); } else { $( "#effect" ).animate({ backgroundColor: "#fff", color: "#000", width: 240 }, 3000 ); } state = !state; }); } );
